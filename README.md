# Fabric-DataPipeline-Demo
Sample project demonstrating data pipelines using Microsoft Fabric.

## Description
This is a demo project to show data pipelines and dashboards using Microsoft Fabric and Azure services. 
The project includes data ingestion, transformation, and visualization.

## Goals
- Demonstrate data ingestion from APIs
- Build data pipelines in Microsoft Fabric
- Create dashboards in Power BI
- Automate tasks using Azure Logic Apps

## Project Structure
- `data/` : sample CSV or JSON files
- `notebooks/` : Python, PySpark or SQL scripts
- `dashboards/` : PBIX files or screenshots of dashboards
- `azure-logic-apps/` : Logic Apps JSON or workflow definitions
- `docs/` : diagrams and documentation

## How to Use
1. Clone this repository: git clone https://github.com/YOUR-USERNAME/Fabric-DataPipeline-Demo.git
2. Open scripts in `notebooks/` and run with your data
3. Open PBIX files in Power BI Desktop to see dashboards
4. Explore `docs/` for pipeline diagrams and explanations

## Skills Showcased
- Microsoft Fabric: Data integration, Lakehouse, Synapse
- Power BI dashboards and reporting
- Python / PySpark for data transformations
- Azure Logic Apps for automation
